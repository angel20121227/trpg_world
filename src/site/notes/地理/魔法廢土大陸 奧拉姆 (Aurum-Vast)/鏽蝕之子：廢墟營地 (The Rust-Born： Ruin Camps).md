# 概述

**「鏽蝕之子：廢墟營地」 (The Rust-Born: Ruin Camps)** 是一群沒有固定領土的亡命之徒。他們是一群精良的騎兵與拾荒者，騎乘著由血肉與廢舊零件拼湊而成的「半機械馬」。他們最顯著的特徵是會佔領並重啟那些「[械靈](../../../世界觀/械靈.md)」遺棄在廢土上的移動城市殘骸（被稱為「遺產」），將其作為臨時的移動堡壘。當據點的資源耗盡或面臨威脅時，他們會毫不留戀地撤離，尋找下一個廢墟，因此在不同時期，他們居住的區域會有不同的臨時名稱。

# 人口

- **規模：** 村莊至小城市 (Variable: Village to Small City)
    
- **數量：** 約 3,000 至 5,000 名流動人口。
    
- **主要種族：** 人類 (70%)、半獸人 (20%)、流亡的[械靈](../../../世界觀/械靈.md) (10%)。
    
- **平均教育水平：** 低（多為野外生存技巧、基礎機械維修與馬術）。
    
- **生活品質：** 窮困潦倒（居住環境極差，充滿油汙與鐵鏽，物資主要依靠掠奪與拾荒）。
    
- **軍人占比：** 80%（在荒野中，除了孩童以外的所有人都是戰士與騎手）。
    

# 特質+災禍

- **特質：廢土的寄居者。** 他們對奧拉姆的地理環境瞭若指掌，擅長在短時間內修復並武裝任何報廢的機械結構。
    
- **災禍：不穩定的居所。** 廢棄的移動城市通常伴隨著嚴重的結構受損或魔力核心洩漏，隨時有崩塌或發生劇烈魔爆的危險。
    

# 政府

- **類型：獨裁政府 (Autocracy) / 霸主統治 (Overlord)**
    
- **領導者：【劫掠頭目：鐵蹄·賈克斯 (Jax Iron-Hoof)】**
    
- **集會場所：【廢墟沙龍】**（通常是廢棄城市中的酒吧或燃料補給站）。
    
- **說明：** 以武力決定領導權。頭目必須能帶領眾人找到豐厚的礦藏或掠奪目標，否則很快就會被取代。
    

# 派系

1. **「鐵蹄獵手」 (The Iron-Hoof Hunters)：**
    
    - **目標：** 訓練與改造半機械馬，執行閃電式的掠奪任務。
        
    - **方式：** 騎乘機械馬襲擊「黑岩礦業」的小型運輸車隊，奪取藥劑與燃料。
        
2. **「廢鐵工程師」 (Scrap-Techs)：**
    
    - **目標：** 讓廢棄的移動城市暫時「動起來」，並維持半機械馬的零件供應。
        
    - **方式：** 搜刮戰場上的零件，進行不計後果的拼湊式維修。
        
3. **「荒野領路人」 (Wild-Guides)：**
    
    - **目標：** 尋找新的廢棄據點，並監視其他三大勢力的動向。
        
    - **方式：** 作為賞金獵人將情報高價賣給「阿勒忒亞」或「黑岩礦業」。
        

# 重要NPC

- **頭目·賈克斯 (Jax)：** 一名雙腿皆為機械義肢的人類，其座騎是一匹體型巨大的半機械重型馬「破壞者」。
    
- **「老鬼」莫瑞 (Old Murray)：** 經驗豐富的廢鐵工程師，據說曾是「克洛諾斯」的技工，因偷竊零件被流放。
    
- **「紅雀」莎拉 (Red Sparrow)：** 頂尖的偵查手，負責帶領拾荒小隊在魔法風暴來襲前尋找掩體。
    

# 地理環境與資源

- **類型：流動式野戰營地。**
    
- **資源：** 他們沒有產出，但擁有極強的「回收能力」。他們能將報廢的晶石彈藥重新裝填，並將魔晶石廢料精煉成廉價的燃料。
    

# 宗教

- **信仰：** 掠奪之神絲卡蒂亞 (60%)、混沌之神法索斯 (30%)、無信仰 (10%)。
    
- **說明：** 崇尚強者生存與即時享樂。
    

# 危險度

- **數值：8/10 (對外人具有攻擊性)**
    
- **說明：** 他們視廢土上的一切為無主物，包括旅行者的裝備。除非你能提供他們急需的零件或高度純化的藥劑。
    

# 修正項

- **腐敗 (+6)：** 沒有法律，只有利益。只要報酬夠高，他們不介意背叛僱主。
    
- **犯罪 (+6)：** 掠奪、鬥毆與走私是他們的日常。
    
- **經濟 (-4)：** 完全依賴掠奪與拾荒。
    
- **秩序 (-6)：** 極度混亂，唯一的規則是不要在據點內朝同伴背後開槍。
    
- **文化 (-5)：** 只有關於機械改良與武器維修的粗糙知識。
    
- **社交 (+2)：** 出乎意料地，他們在沙龍裡對有趣的旅行者相當大方，前提是你得帶上好酒。

# 特色生物：半機械馬 (Cyber-Steeds)

- **來源：** 奧拉姆原生的野馬，在受到「晶石病」侵害後，由鏽蝕之子的工程師將其受損的肢體與器官更換為活塞、齒輪與晶石過濾器。
    
- **特性：**
    
    - **環境抗性：** 牠們的肺部被更換為大型過濾風箱，能直接在魔法風暴中奔跑而不會迅速晶化。
        
    - **混合動力：** 不再依賴大量牧草，而是攝取微量的「低純度魔晶石碎屑」與「粗製機油」作為能量來源。
        
- **戰鬥力：**
    
    - **震盪踢擊：** 機械蹄在擊中目標時能引發微小的「壓力釋放」，足以踢穿輕型裝甲或擊碎感染者的晶簇。
        
    - **蒸汽衝刺：** 瞬間排出過量蒸汽以獲得短暫的爆發速度，這在衝鋒或脫離戰場時非常有效。
        
    - **戰術載具：** 馬鞍兩側通常會懸掛**「長柄爆炸長矛」**或**「單發式重型鐵撬槍」**。這些武器多為一次性或需要繁瑣裝填，強調一擊脫離的打法。
