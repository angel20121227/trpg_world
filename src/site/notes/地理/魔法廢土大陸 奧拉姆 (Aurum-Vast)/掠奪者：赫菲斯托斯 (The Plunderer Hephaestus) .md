# 概述

**「掠奪者：赫菲斯托斯」 (The Plunderer: Hephaestus)** 是一座外觀粗獷、充滿工業美學的巨型履帶式移動工廠。它像是一個巨大的、會移動的鋼鐵採石場，前緣裝有數十組足以粉碎山岩的巨型複合鑽頭。這座城市隸屬於「黑岩礦業聯合（Black Rock Mining United）」，其存在的唯一目的就是以最高的效率挖掘魔晶石。它所經之處，大地會留下深不見底的溝壑與濃煙，被廢土居民視為「大地的收割機」。
在掠奪者：赫菲斯托斯的最底部，還有著企業支持的「[赫菲斯托斯：生化資源部 (Department of Bio-Resources, D.B.R.)](../../../世界觀/赫菲斯托斯：生化資源部%20(Department%20of%20Bio-Resources,%20D.B.R.).md)」

# 人口

- **規模：** 都會 (Metropolis)
    
- **數量：** 約 35,000 名常駐人口（包括大量的契約勞工）。
    
- **主要種族：** 人類 (60%)、獸人/半獸人 (30%)、被僱傭的[械靈](../../../世界觀/械靈.md)工頭 (10%)。
    
- **平均教育水平：** 低（除了高層管理人員與地質工程師外，大多數是只會操作機械的體力勞動者）。
    
- **生活品質：** 貧窮至中產（高層主管享受著奢華的封閉套房，而底層礦工則居住在震動且嘈雜的履帶艙間）。
    
- **軍人占比：** 25%（「黑岩保安隊」，裝備精良的企業私兵，負責鎮壓勞工暴動與清剿礦區原住民）。
    

# 特質+災禍

- **特質：資源壟斷者。** 擁有全大陸最強大的開採技術與物流網絡，控制了亞利亞近四成的魔晶石供應。
    
- **災禍：晶石病溫床。** 由於追求產量而忽視防護，城內底層的晶石病發病率是全大陸最高的。
    

# 政府

- **類型：獨裁政府 (Autocracy) / 霸主統治 (Overlord)**
    
- **領導者：【總監：巴隆·鋼脈 (Superintendent Baron Steelvein)】**
    
- **集會場所：【營運指揮中心】**（充滿屏幕與數據圖表的冷冽空間）。
    
- **說明：** 這裡沒有市民，只有「員工」。所有的權力都集中在總監手中，他只需向企業的董事會負責（或者是他自封的王國）。
    

# 派系

1. **「黑岩保安部」 (Security Division)：**
    
    - **目標：** 確保採礦作業不被干擾，保護公司財產。
        
    - **方式：** 使用武力驅逐「阿勒忒亞」的救援隊或掠奪「[械靈](../../../世界觀/械靈.md)」的礦場資訊。
        
2. **「深井反抗軍」 (The Deep-Well Rebels)：**
    
    - **目標：** 改善勞工待遇，推翻巴隆的獨裁統治。
        
    - **方式：** 破壞採礦鑽頭、暗殺腐敗工頭，並與外界的「餘光組織」勾結。
        
3. **「勘探先鋒」 (Prospecting Pioneers)：**
    
    - **目標：** 尋找高純度的「原初魔晶石」。
        
    - **方式：** 駕駛小型快速偵察車深入未開發的魔法廢土，是極度危險但高報酬的職位。
        

# 重要NPC

- **總監·巴隆 (Baron Steelvein)：** 一名安裝了重型機械義肢的人類，冷酷、貪婪，認為廢土上的弱者只是「消耗品」。
    
- **「鼴鼠」格里格 (Grig the Mole)：** 反抗軍的領袖，一名滿身晶石病疤痕的半獸人，對地底隧道瞭若指掌。
    
- **公關部長·維多利亞 (Victoria)：** 優雅但虛偽的女性，負責向外宣傳公司的「慈善事業」，掩蓋內部的非人道行為。
    

# 地理環境與資源

- **類型：移動採礦與提煉複合體。**
    
- **資源：** 擁有極其龐大的魔晶石原礦儲備。這裡的貿易市場非常巨大，從非法火器到違禁藥品應有盡有。
    

# 宗教

- **信仰：** 獵殺之神絲卡蒂亞 (50%)、無神論 (40%)、豐收之神艾德娜 (10%)。
    
- **說明：** 崇尚強者生存的信條，競爭氣氛極其濃厚。
    

# 危險度

- **數值：7/10 (內部治安) / 9/10 (工業區危險)**
    
- **說明：** 雖然秩序由私兵維持，但工業事故、勞工鬥毆與晶石病急性發作讓這裡隨時可能爆發混亂。
    

# 修正項

- **腐敗 (+5)：** 只要有足夠的晶石，你可以買通任何一名守衛。
    
- **犯罪 (+4)：** 黑市交易、走私與非法賭博在底層區域極為猖獗。
    
- **經濟 (+6)：** 產值驚人，是廢土上最重要的經濟支柱。
    
- **秩序 (+3)：** 僅在軍事層級上有秩序，民生區域則是一片混亂。
    
- **文化 (-3)：** 除了採礦技術外，這裡幾乎沒有任何藝術或學術氛圍。
    
- **社交 (-2)：** 對外來者展現出高度的功利主義，除非你有生意要談，否則沒人會理你。

# 相關企業裝備：【黑岩工業系列】

### 晶石燃爆彈 (Cinder-Blast Charges)

- **類型：** 工業用炸藥。
    
- **用途：** 原本用於炸開礦床，但在戰鬥中是非常可怕的區域性武器。
    
- **效果：** 除了物理爆炸，還會釋放大量足以引發晶石病的粉塵廢氣。
    

### 強化外骨骼「重錘」 (Exoskeleton "Sledge")

- **類型：** 動力裝甲。
    
- **用途：** 讓礦工能搬運沉重的礦石，保安隊則將其改裝，配備了巨大的液壓撞擊錘與防彈盾牌。
    

### 興奮劑：藍色怒火 (Blue Fury)

- **類型：** 戰鬥/工作藥劑。
    
- **作用：** 強制燃燒體內剩餘魔力以獲取爆發力，暫時無視晶石病的痛楚。
    
- **代價：** 藥效過後，晶石病會顯著惡化。
