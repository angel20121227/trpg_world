# 概述

克洛諾斯是一座建築在六組巨型複合履帶上的移動堡壘。它高約八層樓，通體由抗腐蝕的黃銅與鋼鐵鑄造。巨大的煙囪終日排放著淡藍色的魔力廢氣，其蒸汽笛聲能傳播至數十里外，既是遷徙的信號，也是對荒野怪物的威懾。

# 人口

- **規模：** 都會 (Metropolis)
- **數量：** 約 25,000 名常駐人口。
- **主要種族：** [械靈](../../../世界觀/械靈.md) (85%)、矮人 (10%)、人類流放者 (5%)。
- **平均教育水平：** 高（主要集中在工程學、符文物理與地質導航）。
- **生活品質：** 中產（雖然物資受限，但分配極其公平，且能源充足）。
- **軍人占比：** 15%（主要為「護衛旅」，負責清理履帶路徑上的巨獸與驅逐採礦公司的武裝探測隊）。
# 特質+災禍

- **特質：移動的技術聖殿。** 擁有全亞利亞最尖端的魔晶石提純反應爐，是蒸氣技術的發源地。
- **災禍：核心過熱與零件疲勞。** 由於長期在缺乏魔力的廢土遷徙，這座城市的關鍵零件已損耗數百年。一旦主反應爐失效，整座城市將成為荒野中靜止的鐵棺材。
  

# 政府

- **類型：民主 / 地方議會 (Council)**
- **領導者：【大齒輪議會 (The Grand Gear Council)】**
- **集會場所：【藍圖大廳】**（原為移動城市的中央控制室）。
- **說明：** 由七位代表不同職能（動力、導航、採礦、醫療、維護、外交、保衛）的長老組成，每十年重選一次。

# 派系

1. **「尋根者」 (The Root-Seekers)：** * **目標：** 尋找傳說中魔法樹的主根殘骸。
   **方式：** 深入最危險的礦洞，進行非法探勘。
2. **「鋼鐵兄弟會」 (Iron Brotherhood)：** * **目標：** 維護[械靈](../../../世界觀/械靈.md)的獨立地位，反對採礦公司的滲透。
   **方式：** 威脅並武裝驅逐進入領地的拓荒傭兵。
3. **「蒸汽工會」 (The Steam Guild)：** * **目標：** 累積財富並壟斷符文刻印技術。
   **方式：** 透過與外地採礦公司進行祕密貿易，換取稀有金屬零件。


# 重要NPC

- **長老·奧托 (Elder Otto)：** 議會首領，一位身體布滿鏽跡但雙眼閃爍藍光的老年[械靈](../../../世界觀/械靈.md)，負責導航。
- **「火花」莉亞 (Spark Lea)：** 人類與矮人的混血兒，天才技師，負責黑市火器的維護，對新住民較為友善。
- **守衛長·鐵錘 (Captain Iron-Hammer)：** 強硬派[械靈](../../../世界觀/械靈.md)戰士，神器「巴斯提恩之盾」的崇拜者，對任何有機種族抱持高度警戒。

# 地理環境與資源

- **類型：動態採礦樞紐。**
  **資源：** 城市本身就是一座移動的工廠。它停留在大型魔晶石礦床上方時，會降下巨大的鑽頭進行採集。它不生產食物，其物資主要依賴與外界貿易魔晶石與精密的蒸汽零件。

# 宗教

- **信仰：** 無神論 (70%)、歐若崇拜 (20%)、其餘 (10%)。
- **說明：** 大多數械靈不信神，他們將機械邏輯視為宇宙唯一的真理。

# 危險度

- **數值：3/10 (內部) / 8/10 (外部)**
- **說明：** 城內極度秩序與安全，但城外則是無法律的魔法廢土，充滿了法索斯產生的畸變怪。

# 修正項

- **腐敗 (-4)：** [械靈](../../../世界觀/械靈.md)很少有私慾，賄賂幾乎無效。
- **犯罪 (-2)：** 嚴格的計算邏輯與監視，使暴力犯罪極少發生。
- **經濟 (+5)：** 掌握了亞利亞最重要的能源提純技術，富可敵國。
- **秩序 (+6)：** 一切行為準則皆如鐘錶般精確，違法者會被強制驅逐。
- **文化 (+3)：** 擁有豐富的技術文獻，但對外界歷史與藝術興趣缺缺。
- **社交 (-5)：** 對外來者極度排外，除非你能展現出極高的機械天賦或帶來的零件極具價值。
