# 概述

**「進化之繭：新黎明號」 (The Evolution Chrysalis: Neo-Dawn)** 是一座由廢棄巨型採礦平台改裝而成的移動要塞。其外觀不像傳統機械那樣整齊，而是布滿了半透明的藍色晶簇與粗大的加壓管道，看起來像是鋼鐵與礦物融合的「活體」，移動時會發出低沉的脈動聲。這裡居住著「晶體共生者」，他們深信晶石病並非毀滅，而是星系神明賜予亞利亞人的進化洗禮，將病徵轉化為生存優勢。

# 人口

- **規模：** 小城市 (Small City)
    
- **數量：** 約 6,500 名常駐人口。
    
- **主要種族：** 人類感染者 (60%)、獸人突變者 (30%)、其他異變種族 (10%)。
    
- **平均教育水平：** 中等（高度偏向生物變異學、晶體物理學與原始薩滿信仰）。
    
- **生活品質：** 貧窮（居民對物質需求極低，能量來源主要依靠體內晶石與魔力的共振）。
    
- **軍人占比：** 40%（幾乎所有成年感染者皆具備戰鬥能力，身體長出的晶簇即是天生的武器）。
    

# 特質+災禍

- **特質：突變的奇點。** 這裡的居民發展出了獨特的「晶石武技」，能操控體內的結晶進行防禦或射擊。
    
- **災禍：不穩定的生命週期。** 追求進化的代價是極高的暴斃率。城內不時會有人因無法承受能量而徹底「晶化死亡」，其遺骸會被直接銲接在城市外牆上。
    

# 政府

- **類型：秘術團體 (Magical) / 獨裁政府 (Autocracy)**
    
- **領導者：【進化導師：大先知·凱爾索 (The Arch-Prophet, Kelso)】**
    
- **集會場所：【共鳴祭壇】**（位於城市中央，是一個巨大的晶石諧振腔體）。
    
- **說明：** 凱爾索是一名末期感染者，但他奇蹟般地維持了理智，被視為進化的最高典範，其話語即是絕對的律法。
    

# 派系

1. **「晶能武士」 (The Shard-Warriors)：**
    
    - **目標：** 透過戰鬥測試進化的強度，並掠奪高純度的魔晶石。
        
    - **方式：** 作為高風險傭兵受僱於「赫菲斯托斯」，換取新型的實驗性試劑。
        
2. **「融合派」 (The Fusionists)：**
    
    - **目標：** 試圖將機械零件直接植入晶化的肉體，達成肉體、機械與晶石的三位一體。
        
    - **方式：** 經常襲擊小型[械靈](../../../世界觀/械靈.md)隊伍，竊取其身體零件與核心結構進行「身體嫁接」。
        
3. **「石中蟬」 (The Stone Cicadas)：**
    
    - **目標：** 秘密反抗組織。保護尚未完全晶化的孩童或清醒者，將其秘密送往「阿勒忒亞」接受治療。
        
    - **方式：** 利用魔法風暴的掩護進行「人道走私」，並暗中將強效突變劑替換為緩解藥劑以延緩病程。
        

# 重要NPC

- **先知·凱爾索 (Prophet Kelso)：** 全身覆蓋著晶藍色甲冑（實為皮膚變質），能透過晶石振動發出直擊心靈的聲音。
    
- **「碎裂者」巴拉克 (Barak the Shatterer)：** 戰鬥指揮官，雙臂已轉化為巨大的晶石重錘，性格暴躁且極度崇尚武力。
    
- **藥劑師·希拉 (Apothecary Sheila)：** 負責調配各方換來的藥劑，也是「石中蟬」暗中觀察與懷疑的對象。
    
- **「老蟬」尤里 (Old Yuri)：** 「石中蟬」的地下領袖。
    

# 地理環境與資源

- **類型：移動武裝修道院 / 實驗性聚落。**
    
- **資源：** 由於對廢氣的高度耐受性，他們能進入其他勢力無法生存的高濃度汙染區，採集極其罕見的「原始晶核」。
    

# 宗教

- **信仰：** 混沌之神法索斯 (50%)、變革之神澤菲爾 (30%)、自我進化信仰 (20%)。
    
- **說明：** 崇拜法索斯帶來的「不確定性」，認為隨機的突變才是進化的真諦。
    

# 危險度

- **數值：9/10 (對非感染者極度危險)**
    
- **說明：** 這裡不歡迎「純潔者」，外人若非貿易夥伴，極易被視為進化的「活體祭品」。
    

# 修正項

- **腐敗 (+2)：** 對內部教義忠誠，但可用「更強大的進化物質」賄賂其高層。
    
- **犯罪 (+3)：** 內部鬥爭頻繁，強度決定生存權。
    
- **經濟 (-2)：** 幾乎沒有傳統貿易，完全依靠掠奪與藥劑配給支撐。
    
- **秩序 (+1)：** 法律鬆散，僅靠先知的威嚴維持表面的和平。
    
- **文化 (+2)：** 擁有關於變異生理學與晶石共鳴的獨特知識體系。
    
- **社交 (-6)：** 對非我族類抱持極端的排外與侵略性。
