# 概述

**「餘燼榮耀：凱旋號」 (The Embers of Honor: Triumph)** 是一列由十二節巨型重裝甲車廂組成的陸地列車。其外觀呈現嚴謹的軍事深灰色，外殼覆蓋著抗魔反應裝甲與加固鋼板。這支部隊原是某個覆滅國家的正規軍殘部，現轉型為奧拉姆（Aurum-Vast）最可靠的武裝護衛力量。他們不追求進化，也不追求無序的掠奪，唯一的信條是「守護合約、維護軍人的尊嚴，以及團隊的存續」。

# 人口

- **規模：** 小城市 (Small City) / 武裝序列
    
- **數量：** 約 5,000 名常駐成員（含受訓軍人及其家屬）。
    
- **主要種族：** 人類 (70%)、矮人重裝步兵 (20%)、[械靈](../../../世界觀/械靈.md)戰術分析官 (10%)。
    
- **平均教育水平：** 中高（強調戰術素養、通訊工程與武器科學）。
    
- **生活品質：** 中產（物資分配極度精確，環境整潔，但生活受到嚴格的軍事化規律限制）。
    
- **軍人占比：** 70%（高度軍事化的社群，所有成年居民皆需負擔巡邏、整備或後勤維修職責）。
    

# 特質+災禍

- **特質：絕對的契約精神。** 他們擁有奧拉姆最專業的運輸保護紀錄，是各大企業在跨區域運送珍貴物資時的首選護衛。
    
- **災禍：資源供給依賴。** 由於其軍事設備極其依賴標準化彈藥與高品質零件，他們必須持續與「黑岩礦業」或「克洛諾斯」簽署高難度的合約以換取後勤支援。
    

# 政府

- **類型：獨裁政府 (Autocracy) / 軍政府**
    
- **領導者：【大元帥：凱斯勒·馮·布勞恩 (Grand Marshal Kessler von Braun)】**
    
- **集會場所：【作戰指揮車廂】**（配備了全息沙盤與通訊陣列）。
    
- **說明：** 實行嚴格的階級制度，居民的福利與地位直接掛鉤於其軍銜與實戰戰功。
    

# 派系

1. **「先鋒盾衛」 (The Vanguard Shields)：**
    
    - **目標：** 在陸地列車前方開路，應對巨獸攻擊或清理廢墟障礙。
        
    - **方式：** 配備重型盾牌與短程晶體散彈槍，執行防禦性突擊。
        
2. **「後勤軍需部」 (Logistics Corps)：**
    
    - **目標：** 負責陸地列車的維修與物資配給，確保在缺乏零件的廢土中持續前進。
        
    - **方式：** 進行極其精準的資源管理，並研發節能型蒸汽引擎。
        
3. **「契約仲裁辦」 (Contract Arbitration Office)：**
    
    - **目標：** 與各方勢力進行外交談判，確保佣金及時以魔晶石或藥劑形式支付。
        
    - **方式：** 法律談判與武力威懾並行。
        

# 重要NPC

- **大元帥·凱斯勒：** 一名一絲不苟的老軍人，即使在資源匱乏時也維持著整潔的制服。他視「凱旋號」為他守護的最後國土。
    
- **上尉·蕾娜 (Captain Lena)：** 陸地列車的主駕駛員，對「新黎明號」的變異者抱持極大的厭惡，認為他們是生命的背叛。
    
- **技術官·齒輪 (Tactical Officer Gears)：** 一名擅長彈道計算的[械靈](../../../世界觀/械靈.md)，負責指揮要塞的遠程火炮防護系統。
    

# 地理環境與資源

- **類型：移動重型武裝要塞。**
    
- **資源：** 擁有亞利亞大陸最優良的常規武器維護流水線與專業的戰術情報庫。
    

# 宗教

- **信仰：** 守護之神巴斯提恩 (80%)、秩序之神歐若 (20%)。
    
- **說明：** 他們是巴斯提恩的虔誠信徒，強調「守護」與「穩固」的道德價值。
    

# 危險度

- **數值：4/10 (內部極度安全) / 6/10 (對外武力威懾)**
    
- **說明：** 雖然具備強大火力，但除非合約要求，否則他們極少主動攻擊他人。
    

# 修正項

- **腐敗 (-6)：** 賄賂軍人被視為嚴重的叛國罪。
    
- **犯罪 (-6)：** 嚴厲的軍法統治，幾乎沒有內部犯罪。
    
- **經濟 (+2)：** 收入穩定，但主要用於武器保養而非改善奢侈生活。
    
- **秩序 (+10)：** 全亞利亞秩序最高的移動據點。
    
- **文化 (+1)：** 除了軍事歷史與榮譽授勳，幾乎沒有娛樂活動。
    
- **社交 (-3)：** 專業但冷漠，將所有外來者視為潛在的「委託人」或「潛在威脅」。
