### 概述

**凡納爾** 坐落於斷裂的世界樹 **德拉希爾 (Drasill)** 巨大的樹墩斷面上。它是地表上最強大的中立都市，也是連接雲端聖域「塞雷斯提亞」與地表荒野的唯一樞紐。城市由「探索者公會」直接管理，呈現出極其務實、充滿冒險氣息的氛圍。巨大的「天之淚」瀑布從高空落下，穿過凡納爾的中心，為整座城市提供永不枯竭的動力。

### 人口

- **規模：** 巨型都市 (Metropolis)
    
- **數量：** 約 80,000 名常駐人口，每日有數萬名流動冒險者與商人。
    
- **主要種族：** 人類 (45%)、希爾凡精靈 (25%)、矮人 (15%)、[械靈](../../../世界觀/械靈.md) (5%)、其他異族 (10%)。
    

### 特質+災禍

- **特質：探索者的主場。** 這裡的一切規矩由探索者公會制定，只要你有能力完成委託，這裡就是你的天堂。
    
- **災禍：地脈引力異常。** 由於法理之杖的能量偶爾波動，城市內某些區域會出現短暫的引力失效，導致物體或建築碎片漂浮。
    

### 政府

- **類型：專家統治 (Technocracy) / 探索者公會直接管理。**
    
- **領導者：公會長·艾拉 (Guildmaster Elara)。**
    
- **說明：** 實行功績制，社會地位與資源分配取決於在公會中的貢獻度與冒險勳章等級。
    

### 派系

- **探索者公會：** 城市的統治與維安主體，掌握著全大陸最詳盡的地圖與遺跡資料。
    
- **埃忒爾學院評議會：** 懸浮於城市上空的學術機構，雖然政治獨立，但為城市提供關鍵的防禦與戰術技術支援。
    
- **鐵衛軍：** 由公會直接領導的精銳武裝，成員多為學院「戰術系」與「防禦系」的畢業生。
    

### 重要NPC

- **公會長·艾拉 (Elara)：** 希爾凡精靈，性格強勢且務實，她認為生存與探索高於一切。
    
- **大導師·卡斯提安 (Castian)：** 精靈智者，星髓塔的管理者，與艾拉在研究資源與城市軍事化議題上經常針鋒相對。
    
- **「鐵衛」索恩 (Thorn)：** 凡納爾防衛隊長，負責城市武力調度，性格冷靜沉默。
    

### 地理環境與資源

- **類型：世界樹殘骸平台 / 垂直水利樞紐。**
    
- **資源：** 擁有全大陸唯一的「法理之杖」能量接口。利用從世界樹中心噴發的「升騰氣流」，城市具備極佳的垂直運輸能力。
    

### 宗教

- **信仰：德拉希爾崇拜 (40%)、法理之杖崇拜 (30%)、多元宗教 (30%)。**
    
- **說明：** 居民普遍敬畏世界樹的生機，並將法理之杖視為維持物理現實不崩潰的守護神。
    

### 危險度

- **數值：2/10 (市區高度秩序) / 8/10 (地底根系深處或實驗禁區)。**
    
- **說明：** 城市表面治安良好，但深入世界樹內部的根系迷宮仍隱藏著古代寄生獸。
    

### 修正項

- **腐敗 (-4)：** 功績制下，賄賂不如實力有效。
    
- **經濟 (+10)：** 壟斷聖水貿易與情報，財政極其富有。
    
- **文化 (+9)：** 擁有大陸最密集的圖書館與異族沙龍。
    
- **秩序 (+7)：** 法律由簡潔的公會法構成，但由強大的「防禦」魔法與衛兵維持。
    
- **社交 (+10)：** 對所有種族（包含地球探測者）都充滿好奇與包容。

### 星髓塔·埃忒爾 (Aetheris: The Astral Marrow)

懸浮在凡納爾正上方五百公尺處的獨立學術浮島，這座浮島是世界樹頂端崩落的遺骸，被地底「法理之杖」的強大引力鎖定在凡納爾正上方。

- **管理機構：** 學院評議會 (The Faculty Council)
    
- **最高負責人：** **大導師·卡斯提安 (Archmage Castian)** —— 嚴謹的高階精靈，致力於解析大崩裂的真相。
    
- **學術立場：** 享有完全的學術獨立權，但需受聘於探索者公會，為城市提供防禦與軍事技術支持。

#### 核心六大學科 (The Six Disciplines)

| **學科名稱**             | **研究核心**           | **軍事/實務應用**                 |
| -------------------- | ------------------ | --------------------------- |
| **法理 (Axioms)**      | 空間重構、引力、物理常數       | 維持城市結構穩定、重力場壓制術、空間摺疊。       |
| **源泉 (Founts)**      | 聖杯之水、液態瑪娜、鍊金術      | 精準瑪娜灌注、廣域治癒、晶石病緩解劑。         |
| **防禦 (Defense)**     | **魔法防禦、反制術式、結界架構** | 解析並中和敵方術式、建立城市級魔法迷鎖、反法索斯干擾。 |
| **戰術 (Tactics)**     | **軍用魔法、遠程轟炸、戰場導航** | 魔法彈幕覆蓋、單體高能打擊、戰場魔力波段管制。     |
| **遺產 (Legacies)**    | 古符文解密、世界樹歷史、地脈學    | 修復德拉希爾脈絡、回收與重啟古代魔法遺物。       |
| **畸變 (Abberations)** | 空間裂隙、暗精靈研究、法索斯觀測   | 偵測隱形與位面潛伏、監控世界裂縫、研發反蠱惑裝置。   |
