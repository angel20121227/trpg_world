# 概述

塞拉菲亞曾是亞利亞大陸的魔法心臟。在「大崩裂事件」後，支撐這片區域的核心——**世界樹「德拉希爾」**雖然半截枯萎，但其散發的殘餘生機將巨大的地塊撕裂並送入高空，形成了無數的「空島」。這裡空氣中瀰漫著高濃度的液態瑪娜，植物生長速度驚人且具備微弱的神智。
